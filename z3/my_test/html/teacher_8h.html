Method to build a transform a counterexample from z3::expr to a vector of int vectors. Used for existential and universal counterexamples.

Parameters
counterexample- counterexample of the hypothesis encoded as z3::expr
start- This is the counterexample node.
edges- edges of the game graph
context- context to evaluate the expressions
variables- variables to get the value of each variable assigned
variables_dash- variables in the next step
all_variables- variables and variables_dash combined
n- maximum number of successors
Returns
Returns a counterexample as a horn clause. First vector is left side, rest of the vectors are the right side.
